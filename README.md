## House Rental System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/38e954ec829b4ee1ba73854fd2a80a82/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators, property owners.

**Regular User Functions:**
1. Login
2. Registration
3. Personal Center
4. Schedule Property Viewing
5. Contract Management
6. Property Warranty Management
7. Maintenance Processing Management
8. Property Review Management
9. Become a Property Owner
10. My Favorites Management


**Administrator Functions:**
1. Login
2. Personal Center
3. User Management
4. Property Owner Management
5. Property Type Management
6. Property Information Management
7. Property Viewing Appointment Management
8. Contract Information Management
9. Property Warranty Management
10. Maintenance Processing Management
11. Property Review Management
12. Become Property Owner Management
13. Message Board Management
14. System Management

**Property Owner Functions:**
1. Login
2. Personal Center
3. Property Information Management
4. Property Viewing Appointment Management
5. Contract Information Management
6. Property Information Management
7. Property Warranty Management
8. Maintenance Processing Management
9. Property Reviews
